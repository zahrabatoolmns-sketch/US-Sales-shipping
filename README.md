#  Shipping & Supply Chain Analytics Dashboard

A comprehensive Power BI Dashboard designed to analyze and optimize supply chain operations, monitor customer retention, and track social sentiment alongside ROI.

---

##  Dashboard Pages & Key Features

The dashboard consists of three main analytical pages, each focusing on critical business areas:

### 1.  Supply Chain Operations
* **Geographical Mapping:** Track shipping routes, delivery locations, and regional performance using interactive maps.
* **Decomposition Tree:** Deep-dive into root causes of supply chain delays, costs, and operational bottlenecks.
* **Advanced Slicers:** Filter data dynamically by date ranges, carriers, product types, and regions.

### 2.  Customer Retention Analysis
* **Retention Trends:** Visualize customer loyalty and repeat purchase behavior using optimized column charts.
* **Key Metrics (KPI Cards):** Instant visibility into total customers, churn rate, and customer lifetime value (CLV).
* **Segment Slicers:** Filter retention metrics by demographics or customer tiers.

### 3.  Social Sentiment & ROI Tracking
* **Sentiment Breakdown:** Analyze customer feedback and social media sentiment using treemaps.
* **Financial ROI Tracking:** Monitor campaign performance and operational ROI over time with line charts.
* **Executive Summary Textbox:** Contextual insights and recommendations baked directly into the report.

---

##  Tech Stack & Tools Used
* **Power BI Desktop:** For data modeling, DAX calculations, and report visualization.
* **Power Query:** Used for ETL (Extract, Transform, Load) processes to clean and shape the shipping dataset.
* **Data Sources:** Shipping logs, customer feedback data, and financial records.

---

##  How to Use
1. Download the `shipping.pbix` file from this repository.
2. Open the file using **Power BI Desktop**.
3. If necessary, update the data source paths in **Transform Data > Data Source Settings**.
4. Click **Refresh** to load the latest shipping and retention data.

---

##  Key Insights Addressed
* Where are the primary bottlenecks in the current supply chain?
* Which customer segments show the highest retention rates?
